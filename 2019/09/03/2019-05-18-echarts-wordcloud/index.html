echarts字符云图词云图

echarts3也支持字符云了

最近在用echarts2做字符云效果，遇到了一些问题，在网上找解决方案时候偶然间发现echarts3居然也可以做字符云效果了

效果：

之前用echarts3做字符云效果（type : ‘wordCloud’）都不支持，现在只要我们多导入一个worldcloud.js库就可以用echarts3做字符云了

点击查看：worldcloud.js
引入echarts后引入词云图代码词云核心代码

代码:

myChart.setOption({
        title: {
            text: 'echarts3字符云图展示'
        },
        tooltip: {},
        series: [{
            type : 'wordCloud',  //类型为字符云
            shape:'smooth',  //平滑
            gridSize : 2, //网格尺寸
            size : ['80%','80%'],
            //sizeRange : [ 50, 100 ],
            rotationRange : [ 46, 80 ], //旋转范围
            data:[
              {"name": "岁的佛山大佛","value": "200"},
              {"name": "刚刚","value": "156"},
              {"name": "斯蒂芬","value": "122"},
